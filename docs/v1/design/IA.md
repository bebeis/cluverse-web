# (1) IA / 화면 설계 목록 + 화면별 상태값 (v1.1)

## 1.1 앱 IA (모바일 기준: Bottom Tab 5개)

### Tab 1) 홈(Home)

* 홈 피드(구독 보드 + 인기/추천 혼합)
* 트렌딩(랭킹)
* 팔로잉 피드(팔로우한 유저 활동 모음)

### Tab 2) 탐색(Explore)

* 보드 디렉토리(학과/관심사)
* 그룹 디렉토리(카테고리/태그/지역/온오프)
* **검색 UI는 와이어프레임에 포함 가능하나 API_V1 제외 → “준비중” 처리**

### Tab 3) 그룹(Groups)

* 내 그룹 목록(멤버/운영)
* 내 지원 내역(제출됨/검토중/승인/거절)
* 그룹 생성 / 모집 관리(오너/관리자)

### Tab 4) 알림(Notifications)

* 댓글/답글/승인결과/지원도착/공지/팔로우 활동 알림

### Tab 5) 마이(My)

* 내 프로필(공개/비공개/필드 공개 설정)
* 전공 관리(주/복수/부전공)
* 관심 태그 관리(구조화)
* 내 글/내 댓글/북마크
* 차단 목록
* 인증 상태/재시도
* 설정(알림/약관/문의/탈퇴)

## 1.2 화면 목록 (Screen ID) — MVP v1.1

### A. 온보딩/인증/계정

| Screen ID | 화면명           | 접근     | 핵심 기능               |
| --------- | ------------- | ------ | ------------------- |
| A-01      | 로그인/가입 선택     | Guest  | 이메일/소셜 로그인, 회원가입    |
| A-02      | 회원가입          | Guest  | 이메일/비번, 약관 동의       |
| A-03      | 학생 인증 안내      | Member | 인증 필요 기능 안내         |
| A-04      | 학교 이메일 인증     | Member | 이메일 입력→코드 발송/확인     |
| A-05      | 대체 인증 제출      | Member | 학생증/재학증명 업로드(검수)    |
| A-06      | 온보딩: 학교 선택    | Member | 학교 검색/선택            |
| A-07      | 온보딩: 전공 설정    | Member | 주전공 필수 + 복수/부전공 선택  |
| A-08      | 온보딩: 관심 태그 선택 | Member | 구조화 태그 선택(최소 N개 권장) |
| A-09      | 온보딩 완료        | Member | 홈 진입, 인증 유도 배너      |

**상태값**

* Loading: 코드 발송/확인 중, 업로드 중, 검수 제출 중
* Error: 도메인 미지원, 코드 만료/불일치, 파일 형식/용량 초과
* Empty: 학교/학과 검색 결과 없음(“추가 요청” CTA)

### B. 홈/피드/보드/글

| Screen ID | 화면명         | 접근              | 핵심 기능               |
| --------- | ----------- | --------------- |---------------------|
| B-01      | 홈 피드        | 모두(쓰기=Verified) | 카드 피드, 필터(전체/구독/추천) |
| B-02      | 트렌딩         | 모두              | 기간별 랭킹(24h/7d 등)    |
| B-03      | 팔로잉 피드      | Member+         | 팔로우 유저 활동 모음        |
| B-04      | 보드 목록(디렉토리) | 모두              | 학과/관심사, 3-depth 탐색  |
| B-05      | 보드 상세(피드)   | 모두(쓰기=Verified) | 정렬(최신/조회/댓글), 필터    |
| B-06      | 글 상세        | 모두              | 본문/이미지/지표/댓글 트리     |
| B-07      | 글 작성/편집     | Verified        | 카테고리/태그/익명/이미지      |
| B-08      | 북마크 목록      | Member+         | 저장글 목록              |

**핵심 상태값**

* Empty: 보드 글 없음(“첫 글 작성”), 댓글 없음
* 제한: 미인증 쓰기 시 “인증하고 작성”
* 블라인드/삭제: “정책 위반으로 숨김/삭제됨”
* 차단 사용자: “차단한 사용자 콘텐츠(접힘/숨김)”
* 댓글 무한 depth:

    * 기본은 일부 depth만 펼침
    * “대댓글 더보기(서브트리 로딩)”

### C. 댓글 트리(무한 depth UX)

| Screen ID | 화면명           | 접근       | 핵심 기능              |
| --------- | ------------- | -------- | ------------------ |
| C-01      | 댓글 입력(인라인)    | Verified | 댓글/대댓글 작성          |

**상태값**

* Loading: 서브트리 로딩, “더보기” 로딩
* Error: 권한 없음/네트워크
* 제한: 차단 대상 댓글은 접힘 + “보기” 제한(정책)

### F. 그룹/모집/지원/지원자 채팅/전용 게시판

| Screen ID | 화면명          | 접근                | 핵심 기능           |
| --------- | ------------ | ----------------- | --------------- |
| F-01      | 내 그룹         | Verified          | 가입 그룹/운영 그룹 목록  |
| F-02      | 그룹 생성        | Verified          | 카테고리, 태그, 공개범위  |
| F-03      | 그룹 상세(외부)    | 모두                | 소개/모집글/가입(지원)   |
| F-04      | 모집글 작성/편집    | Owner/Admin       | 포지션/조건/기간/마감/질문 |
| F-05      | 모집글 상세       | 모두                | 지원 CTA          |
| F-06      | 지원서 작성       | Verified          | 질문 3~5개 + 링크    |
| F-07      | 지원자 관리       | Owner/Admin       | 제출됨→검토중→승인/거절   |
| F-08      | **지원자 채팅**   | (지원자/Owner/Admin) | 지원 건 단위 Q&A 채팅  |
| F-09      | 그룹 내부 홈(게시판) | Member+           | 공지/일반 탭, 멤버 목록  |
| F-10      | 그룹 게시글 작성    | Member+           | 공지(관리자), 일반(멤버) |
| F-11      | 그룹 게시글 상세    | 권한에 따라            | **외부 공개 옵션** 반영 |
| F-12      | 멤버 관리        | Owner/Admin       | 강퇴/관리자 임명/오너 변경 |
| F-13      | 커스텀 직책 관리    | Owner/Admin       | 직책 생성/부여/회수     |

**그룹 상태값**

* visibility:

    * PUBLIC: 외부 상세/모집 노출
    * PARTIAL: 외부 소개/모집 노출, 내부 게시판 멤버만
    * PRIVATE: 초대 링크 없으면 접근 제한
* 게시글 외부 공개:

    * 그룹 PUBLIC일 때만 “외부 공개” 허용
    * PARTIAL/PRIVATE는 외부 공개 토글 비활성 + 안내

1. 지원 동기

안녕하세요. 평소 알고리즘 문제 해결과 웹 개발에 관심이 많아 혼자 공부하던 중, Cluverse 코딩 동아리의 활동 내용을 보고 지원하게 되었습니다. 특히 지난 학기에 진행하신 '학교 생활 도우미 앱' 프로젝트가 매우 인상 깊었습니다. 혼자 공부할 때 느꼈던 한계를 극복하고, 동료들과 함께 협업하며 실질적인 서비스를 만들어보는 경험을 하고 싶습니다. 성실하게 활동하며 동아리에 기여할 수 있도록 노력하겠습니다.

2. 프로젝트 경험

[개인 프로젝트: To-Do List 웹앱]
- 사용 기술: React, Firebase
- 주요 기능: 할 일 추가/삭제/수정, 드래그 앤 드롭 정렬
- 배운 점: React의 State 관리와 컴포넌트 생명주기에 대해 이해하게 되었습니다.

[팀 프로젝트: 학과 홈페이지 리뉴얼]
- 역할: 프론트엔드 개발 (메인 페이지 UI 구현)
- 성과: 기존 대비 로딩 속도 20% 개선

3. 활동 가능 시간

화요일, 목요일 오후 6시 이후, 주말 자유롭게 가능합니다. 시험 기간 2주 전을 제외하고는 동아리 활동을 최우선으로 하겠습니다.
